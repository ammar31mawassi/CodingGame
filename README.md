# Code Escape

Code Escape is a JavaFX educational puzzle game skeleton. The finished MVP will teach Java basics through three escape-room levels: variables, if-statements, and classes.

## Current Status

The project is organized into app, model, engine, validation, UI, utility, resource, and test packages. The next real implementation step is the validation engine.

## Run

```bash
./mvnw javafx:run
```

On Windows:

```bash
mvnw.cmd javafx:run
```

## Build Order

1. Validation engine
2. Token puzzle screen
3. Player movement and token collection
4. Door unlock and level completion
5. Three-level progression
6. Visual polish and README/demo assets
